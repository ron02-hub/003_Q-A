# EV走行音アンケート データ分析計画書

## 1. 概要

### 1.1 目的
本データ分析計画書は、100名分のサンプルデータから有意義な知見を抽出するための分析手法、可視化方法、およびレポート作成計画を定義する。

### 1.2 分析目標
1. **走行音の印象評価特性の把握**: 各走行音サンプルの評価傾向を明らかにする
2. **購買意欲への影響要因の特定**: 走行音評価と購買意欲の関連性を分析
3. **セグメント別の嗜好分析**: 属性別の走行音嗜好パターンを特定
4. **理想の走行音像の構築**: ラダリング結果から理想的な走行音の要件を導出

## 3. 分析項目詳細

### 3.2 Layer 2: 比較分析

#### 3.2.1 サンプル間比較

| 分析項目 | 手法 | 検定方法 |
|----------|------|----------|
| SD評価の差異 | 対応のある分散分析 | 反復測定ANOVA |
| 購買意欲の差異 | Friedman検定 | ノンパラメトリック |
| WTPの差異 | Friedman検定 | ノンパラメトリック |
| WTPのグループ間差異 | 分散分析（ANOVA） | 属性別グループ間比較 |

#### 3.2.2 最良・最悪音の分析

| 分析項目 | 手法 | 出力形式 |
|----------|------|----------|
| 選択頻度 | 度数集計 | 棒グラフ |
| 選択理由分布 | クロス集計 | ヒートマップ |
| SD評価との関連 | 判別分析 | 散布図 |

### 3.3 Layer 3: 相関・因果分析

#### 3.3.1 相関分析

| 分析項目 | 手法 | 出力形式 |
|----------|------|----------|
| SD軸間相関 | Pearson相関 | 相関行列ヒートマップ |
| SD評価-購買意欲相関 | Spearman相関 | 散布図行列 |
| 属性-評価相関 | 点双列相関 | 相関テーブル |

#### 3.3.2 回帰分析

|----------|----------|------|
| 購買意欲 | SD評価9軸 | 重回帰分析 |
| WTP | SD評価 + 購買意欲 | 順序回帰分析 |
| 最良音選択 | SD評価 | ロジスティック回帰 |

### 3.4 Layer 4: セグメント分析

#### 3.4.1 属性別セグメント

| セグメント軸 | 分析内容 |
|--------------|----------|
| 年齢層別 | SD評価プロファイル、購買意欲差異 |
| 性別 | 嗜好傾向の差異 |
| EV経験別 | 評価基準の違い |
| 音感度別 | 評価の厳しさ/寛容さ |

#### 3.4.2 クラスター分析

| 手法 | 目的 | 期待されるセグメント |
|------|------|----------------------|
| K-means | 評価パターンによる分類 | 3-5クラスター |
| 階層的クラスター | 回答者グループ化 | デンドログラム |

**想定セグメント例:**
1. **静音重視派**: 静粛性を最重視
2. **高級感重視派**: 高級感・品質感を重視
3. **先進性重視派**: EVらしさ・先進性を重視
4. **バランス派**: 全体的なバランスを重視
5. **こだわりなし派**: 走行音への関心が低い
