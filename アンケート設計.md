# EV走行音アンケートアプリケーション アンケート設計書

## 1. 概要

### 1.1 アンケートの目的
EV走行音のユーザー印象を多角的に評価し、購買意欲向上につながるサウンドデザインの方向性を明確化する。

### 1.2 設計思想
**「サウンドデザインによるEVの価値最大化」**を実現するための基礎データ収集

### 1.3 対象者
- 日本国内の成人（20〜70歳）
- 運転免許保有者

### 1.4 所要時間
約45分

---

## 2. 3層構造アプローチ

```
┌─────────────────────────────────────────────────────────────┐
│     Layer 1: 市場価値の定量化（Quantitative Value）          │

└─────────────────────────────────────────────────────────────┘
                        ↓

│     Layer 2: 価値構造の特定（Value Structure）               │

└─────────────────────────────────────────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────────────────┐
│     Layer 3: 心理的文脈の深掘り（Psychological Context）     │

└─────────────────────────────────────────────────────────────┘
```

### 4.1 同意取得

**表示テキスト:**

**同意項目:**
- [ ] 調査への参加に同意します
- [ ] データが匿名化された上で研究目的に使用されることに同意します

### 4.2 基本属性

| 項目 | 入力形式 | 選択肢/範囲 |

| 性別 | ラジオボタン | 男性, 女性, その他, 回答しない |

### 4.3 運転経験

| 項目 | 入力形式 | 選択肢/範囲 |
|------|----------|-------------|
| 運転歴 | スライダー | 0〜50年 |
| EV所有経験 | ラジオボタン | はい / いいえ |

| 項目 | 入力形式 | 選択肢/範囲 |
|------|----------|-------------|
| 周囲の音への感度 | スライダー | 1（気にしない）〜 5（とても気にする） |

### 4.5 音声チェック

**手順:**
1. ヘッドホン/イヤホン使用の確認
2. テスト音声（猫の鳴き声）を再生
3. 「何の音でしたか？」の選択肢形式で確認

**選択肢:**
- 猫の鳴き声 ✓（正解）
- 犬の鳴き声
- 鳥のさえずり
- 車のエンジン音
- 雨の音

**使用音声ファイル:** `99_Material/00_Test/猫の鳴き声1.mp3`

| 2 | 質感 | ざらざら | 滑らか | 音質評価 |
| 3 | 快感情 | 不快 | 心地よい | 情動モデル |
| 4 | 覚醒 | 退屈 | ワクワク | Russell円環モデル |

| 6 | 先進性 | 古臭い | 先進的 | 技術受容 |
| 7 | パワー感 | 弱々しい | 力強い | 車両評価 |
| 8 | 安心感 | 不安 | 安心 | 安全性知覚 |
| 9 | 自然さ | 人工的 | 自然 | 聴覚心理学 |

```
うるさい ───────────┬───────────── 静か
          -3  -2  -1  0  +1  +2  +3

```

### 5.3 購買意欲評価

**提示条件（車両プロファイル）:**
- 外観: Honda N-Box
- 価格: 200万円
- 燃費: 20.0km/L（WLTCモード）

**質問文:**
「この走行音を持つ車を購入したいと思いますか？」

**7段階リッカート尺度:**
1. 全く購入したくない
2. あまり購入したくない
3. どちらかといえば購入したくない
4. どちらでもない
5. どちらかといえば購入したい
6. 購入したい
7. 非常に購入したい

### 5.4 価格受容性評価（WTP）

**質問文:**
「この走行音が理想的だとしたら、車両価格（200万円）に対して、さらにいくらまでなら追加で支払えますか？」

3. 3万円まで

5. 10万円まで
6. 20万円まで

**質問文:**
